1. What is the difference between var, let, and const?
Ans:  
  Var:  Var is used to create variable. It can be updated and redeclared.
  it is old way to use then others.

  Let:  let is a modern technique to use variable. It can be updated, but it can't be declared like var.

  Const: Const is also used as variable. Although it can't be redeclared and updated like others. Used for values that should not change.

3.  What is the difference between map(), filter(), and forEach()?
Ans: 
   forEach(): It is used in an array to execute a function for every element. But it doesn't return a new array. It is mainly used for displaying data, updating values.

   map(): This function creates and returns new array by transforming or modifying each element.

   filter(): This refers to a function which creates new array only when the elements meets the specified condition.
